    DropDocs

  • Get File

    Returns the first 140 characters of a file.

    fileName - path to the file from the root folder

    
						get file myFile.txt
						myFile.txt:
    Locavore leggings quinoa biodiesel chia consequat. Shabby chic ullamco duis, stumptown next level Austin kale chips fingerstache chillwave.

  • Create File

    Creates a file with or without content.

    fileName - path to the file from the root folder

    content (optional) - content for the file

    
						add file HelloWorld.txt Hello, World!
						get file HelloWorld.txt
						
						File HelloWorld.txt created!
						HelloWorld.txt:
						Hello, World!

  • Delete File

    Remove a file from your Dropbox folder.

    fileName - path to the file from the root folder

    
						delete file HelloWorld.txt
						
						HelloWorld.txt successfully deleted!

  • Create Folder

    Creates a folder.

    folderName - path to the folder from the root folder

    
						add folder MyDocs
						
						Folder MyDocs created!

  • Delete Folder

    Remove a folder from your Dropbox folder.

    folderName - path to the folder from the root folder

    
						delete folder MyDocs
						
						MyDocs successfully deleted!
